# Liquor Sales ERP

A complete Liquor Sales ERP developed in VB.NET, SQLite and Crystal Reports.

## Features

- Sales Entry
- Purchase Entry
- Stock Management
- Party Ledger
- Cash & Bank Book
- Expense Management
- Daily Sale Report
- Profit & Loss Report
- Purchase vs Sales Report
- Current Stock Report
- Crystal Reports
- Automatic SQLite Database Creation
- Multi Financial Year Support
- User Login & Company Management
- Multi Company Creation Support
- 
## Technologies Used

- VB.NET (.NET Framework 4.8)
- SQLite Database
- Crystal Reports
- Visual Studio 2026

## Project Structure

- Forms
- Modules
- Reports
- Database
- Images

## Requirements

- Windows 10/11
- .NET Framework 4.8
- SAP Crystal Reports Runtime
- SQLite

## Installation

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Build the project.
4. Run the application.

## Database

The application automatically creates the SQLite database (`INV.db`) if it does not exist.
